# Frame-difference-method
帧差法检测运动车辆抛洒物

优化策略：

1、尝试最基础的帧差法

   结果见diff_videos/1.mp4。
   
2、尝试加入邻接最低帧差法（自定义：即取周围一片的最低帧差）

   取border=1，结果见diff_videos/2.mp4。

3、改变策略。

   使用支持集图片与目标视频的帧差作为输出。（原来是采用前后帧差进行的）

4、优化3的结果

   腐蚀、膨胀、二值化、中值滤波等办法。
   
5、针对4的结果，进行目标检测

   检测出红框加到原视频上输出。
